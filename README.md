# Inventory Management System

A modern Inventory Management System built with Laravel 12 and Filament 4, designed for managing products, stock movements, suppliers, and categories with role-based access control.

---

## 🚀 Overview

This project simulates a real-world inventory system used in internal business operations.
It focuses on clean architecture, authorization, and practical inventory workflows such as stock in/out tracking.

---

## ✨ Key Features

### 📦 Product Management

- Create, update, and delete products
- Automatic SKU generation
- Product image upload (Cloudinary)
- Stock & minimum stock tracking

### 🔄 Stock Management

- Stock In / Stock Out system
- Full stock movement history per product
- Real-time stock updates

### 🏷️ Category & Supplier Management

- Organize products by category
- Manage supplier data and relationships

### 🔐 Role-Based Access Control

- Admin & Staff roles
- Policy-based authorization
- Restricted actions for non-admin users

### ☁️ Cloud Image Storage

- Product images stored in Cloudinary
- No local storage dependency for media

### 📊 Admin Dashboard (Filament)

- Clean and structured admin panel
- Table, form, and detail views
- Relation manager for stock history

---
